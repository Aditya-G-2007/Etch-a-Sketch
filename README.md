# Etch-a-Sketch 🎨

A browser-based, interactive drawing board that allows users to create pixel art directly in their browser. Built with a focus on **dynamic DOM manipulation** and flexible grid layouts.

## ✨ Features
* **Dynamic Grid:** A clean white canvas ready for pixel-perfect designs.
* **Multiple Brush Modes:**
    * **Black:** Classic solid ink.
    * **Gray:** For subtle shading and depth.
    * **Random (Rainbow):** Generates a random color for every pixel touched for a vibrant effect.
* **Eraser:** Precision tool to fix mistakes or refine your art.
* **Reset:** Clear the entire board with a single click to start fresh.

---

## 🛠️ Built With
* **HTML5:** Structure of the drawing board and controls.
* **CSS3:** Flexbox for layout and custom styling (purple/red theme).
* **JavaScript (ES6):** Powering the drawing logic, grid generation, and event handling via the **DOM**.

---

## 📖 How to Use
1. **Select a tool** from the control panel at the bottom.
2. **Hover or click** (depending on your implementation) over the white grid to start drawing.
3. **Use the Random button** for a multi-color experience.
4. **If you want to start over**, hit the **Reset** button!
